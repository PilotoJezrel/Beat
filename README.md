# BEAT

# Features
* Listen to your favorite music
* Message other artist
* Rate others music
* Search other music or artist

* Database Design using Appwrite
* User Registration & Authenticaiton
* Deployment

# Tech Stack
* React Native
* AppWrite
* HTML, CSS and JS
